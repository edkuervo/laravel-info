Instalación de Laravel

Verificar la instalación de composer se ejecuta en la consola: composer

Para una instalación de laravel de una versión específica con su última actualización ejecutamos el comando

composer create-project laravel/laravel miproyecto "7.*"

O una versión más específica agregamos la actualización

composer create-project laravel/laravel miproyecto "7.2.*"

Para verificar la versión de un projecto de Laravel usamos el comando en la carpeta de la instalación

php artisan --version

Ejecutar el servidor

Para lanzar el servidor que tiene Laravel por defecto, podemos usar el comando

php artisan serve

Y para detener la ejecución en la consola de comandos presionamos las teclas Ctrl + C

Crear migración

Las migraciones son las que se encargan de definir la estructura de las tablas que se van a crear en la base de datos, su nombre debe ser en ingles y en plural

php artisan make:migration [name_of_the_table]

Alimentar la base de datos con Seeders

Para crear un seeder, usamos el comando

php artisan make:seeder ProfessionSeeder

Para ejecutar los seeders usamos el comando

php artisan db:seed

Crear modelos

Los modelos se usan para interactuar con la base de datos, y están ubicados dentro de la carpeta app/Models. El nombre sería en singular y relacionado con el nombre de la base de datos

php artisan make:model [Modelo]

Rutas

Las rutas normalmente las creamos/editamos en wl archivo web.php que encontramos en el directorio routes
